# NewsSphere 🌐📰

*NewsSphere* is a modern, responsive news aggregator web application that fetches the latest news from multiple categories and displays them in an interactive, visually appealing interface. It features *Glassmorphism UI*, dark mode, smooth sliders, and a responsive design for all devices.

---

## Features
- *Category-Based News:* Filter by General, Business, Sports, Entertainment, Technology.  
- *Search Functionality:* Quickly search for news topics.  
- *Glassmorphism UI:* Frosted glass effect for cards, sliders, and navigation.  
- *Responsive Design:* Optimized for desktop, tablet, and mobile devices.  
- *Dark Mode Toggle:* Switch between light and dark themes.  
- *Breaking News Slider:* Highlights trending news in a horizontal scrolling slider.  
- *Interactive News Cards:* Each card has an image, title, snippet, and "Read More" link.
